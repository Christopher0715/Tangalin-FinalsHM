Language that we learned

CSS

Cascading Style Sheets (CSS) is a language for describing the appearance of a document written in a markup language like HTML. Along with HTML and JavaScript, CSS is a key component of the World Wide Web.

Java

Java is an object-oriented programming language with a high level of abstraction and as few implementation dependencies as possible.

HTML

HTML, or HyperText Markup Language, is the standard markup language for texts that are intended to be viewed on a web browser. Technologies such as Cascading Style Sheets and scripting languages like JavaScript can help.
